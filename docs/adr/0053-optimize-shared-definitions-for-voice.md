# Optimize shared definitions for voice

Cognidesk will support voice-optimized Journeys, Tools, and Knowledge through channel-specific metadata and instruction overrides on the same shared definitions rather than duplicating parallel voice-only capabilities. The Flight Demo voice agent should keep capability parity with the chat agent while tuning examples, field collection prompts, state instructions, and retrieval behavior so the same booking, status, baggage, disruption, and handoff capabilities work naturally in spoken conversation.
