# Limit v1 OpenAI voice to gpt-realtime-2

`@cognidesk/integration-voice-openai` will support `gpt-realtime-2` as the only realtime voice model for v1. This keeps the first Voice Model Set and prompt/session behavior focused on one voice-optimized model, leaving broader realtime model selection as a future compatibility and evaluation decision rather than an adapter escape hatch in the initial release.
