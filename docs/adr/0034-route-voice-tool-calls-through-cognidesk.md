# Route voice tool calls through Cognidesk

Realtime voice models may see provider-facing Voice Tool Projections, but Cognidesk will execute the underlying Tools through the existing server-side runtime pipeline. This keeps Tool Scope, side-effect confirmation, idempotency, Runtime Events, telemetry, privacy hooks, and Journey Policy authoritative across chat and voice instead of letting browser clients, telephony adapters, or provider connectors bypass SDK behavior.
