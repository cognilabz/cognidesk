# Ship default voice instructions

Cognidesk will ship a Default Voice Instruction Layer for v1 Voice Profiles so developers can enable `gpt-realtime-2` voice without authoring a complete spoken prompt from scratch. Applications may override or extend the default for brand tone and domain phrasing, but the SDK default should cover concise spoken responses, no visual formatting, side-effect confirmation, interruption recovery, and voice-to-chat continuation for detailed content.
