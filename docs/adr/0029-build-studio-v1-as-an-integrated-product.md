# Build Studio v1 as an integrated product

Cognidesk Studio v1 will be implemented as one integrated product slice rather than shipping a read-only Studio first and adding Studio Operator capabilities later. The implementation may be built through internal vertical slices, such as Studio shell/auth/RBAC/database/manifest, Studio Adapter and Agent View, telemetry dashboards, artifact storage and dashboard sandbox/data layer, and Studio Operator Runtime sessions/sandboxes/credential grants/pull requests. These slices are implementation structure, not separate product goals: the v1 product goal includes Agent View, monitoring dashboards, Studio Dashboard generation and publication, and Studio Operator code-change workflows together so the core Studio experience is validated end to end.
