# Use Tailwind-based UI with precompiled CSS

Cognidesk UI will be authored with Tailwind conventions and require Tailwind as part of the design approach, but the package will ship precompiled CSS so consumers can use default components without configuring Tailwind to scan Cognidesk internals. Customization remains available through Appearance Configuration, stable Element Keys, Tailwind class overrides, CSS variables, and headless components.
