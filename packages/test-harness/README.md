# @cognidesk/test-harness

Test harness utilities for Cognidesk runtime and provider package conformance.

## Install

```bash
npm install @cognidesk/test-harness
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/test-harness";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.


## Public exports

- `@cognidesk/test-harness`
- `@cognidesk/test-harness/cli`
- `@cognidesk/test-harness/providers`
