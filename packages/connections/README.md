# @cognidesk/connections

Connection-backed provider helpers for reviewed OpenAPI and MCP provider surfaces.

## Install

```bash
npm install @cognidesk/connections
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/connections";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
