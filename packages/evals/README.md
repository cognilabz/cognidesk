# @cognidesk/evals

Evaluation helpers for testing Cognidesk agents, journeys, and support behavior.

## Install

```bash
npm install @cognidesk/evals
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/evals";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
