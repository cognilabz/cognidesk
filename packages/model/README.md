# @cognidesk/model

Model provider adapters and model-set helpers for Cognidesk runtimes.

## Install

```bash
npm install @cognidesk/model
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/model";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
