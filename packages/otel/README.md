# @cognidesk/otel

OpenTelemetry instrumentation for Cognidesk runtime spans, metrics, and traces.

## Install

```bash
npm install @cognidesk/otel
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/otel";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
