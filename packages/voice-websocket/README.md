# @cognidesk/voice-websocket

Voice WebSocket adapter utilities for Cognidesk realtime voice sessions.

## Install

```bash
npm install @cognidesk/voice-websocket
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/voice-websocket";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
