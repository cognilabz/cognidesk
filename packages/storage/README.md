# @cognidesk/storage

Storage contracts and built-in storage adapters for Cognidesk runtime state.

## Install

```bash
npm install @cognidesk/storage
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/storage";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.


## Public exports

- `@cognidesk/storage`
- `@cognidesk/storage/conformance`
- `@cognidesk/storage/manifests`
- `@cognidesk/storage/postgres`
- `@cognidesk/storage/sqlite`
