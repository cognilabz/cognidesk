# @cognidesk/integration-catalog

Generated integration catalog metadata for Cognidesk Studio and provider discovery.

## Install

```bash
npm install @cognidesk/integration-catalog
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-catalog";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
