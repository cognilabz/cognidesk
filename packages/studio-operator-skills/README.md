# @cognidesk/studio-operator-skills

Operator skill definitions and helpers for Cognidesk Studio automation workflows.

## Install

```bash
npm install @cognidesk/studio-operator-skills
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/studio-operator-skills";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
