# @cognidesk/integration-kit

Shared helpers for building Cognidesk provider integration packages.

## Install

```bash
npm install @cognidesk/integration-kit
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-kit";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
