# @cognidesk/studio-contracts

Shared Studio API schemas, auth helpers, and contract types.

## Install

```bash
npm install @cognidesk/studio-contracts
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/studio-contracts";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.


## Public exports

- `@cognidesk/studio-contracts`
- `@cognidesk/studio-contracts/runtime-auth`
