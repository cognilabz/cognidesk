# @cognidesk/journey-index-json

JSON-backed journey index utilities for loading and publishing journey metadata.

## Install

```bash
npm install @cognidesk/journey-index-json
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/journey-index-json";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
