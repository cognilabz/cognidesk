# @cognidesk/studio-adapter

Target-side Studio adapter for exposing Cognidesk runtime introspection and conversation data.

## Install

```bash
npm install @cognidesk/studio-adapter
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/studio-adapter";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.
