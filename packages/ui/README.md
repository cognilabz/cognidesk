# @cognidesk/ui

Shared Cognidesk UI primitives for Studio and runtime-facing interfaces.

## Install

```bash
npm install @cognidesk/ui
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/ui";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.


## Public exports

- `@cognidesk/ui`
- `@cognidesk/ui/styles.css`
