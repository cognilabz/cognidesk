# @cognidesk/http

HTTP and SSE transport adapters for hosting Cognidesk runtimes.

## Install

```bash
npm install @cognidesk/http
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/http";
```

This package is part of the Cognidesk SDK. It is designed to be composed with the other `@cognidesk/*` packages in application-owned runtimes, transports, storage, and user interfaces.


## Public exports
