# @cognidesk/integration-cobrowsing-cognidesk

Cognidesk cobrowsing integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-cobrowsing-cognidesk
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-cobrowsing-cognidesk";
```

This package exposes the reviewed Cognidesk cobrowsing support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-cobrowsing-cognidesk/manifest";
```
