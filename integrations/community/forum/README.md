# @cognidesk/integration-community-forum

Forum community integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-community-forum
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-community-forum";
```

This package exposes the reviewed Forum community support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-community-forum/manifest";
```
