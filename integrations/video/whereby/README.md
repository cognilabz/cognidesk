# @cognidesk/integration-video-whereby

Whereby video integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-video-whereby
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-video-whereby";
```

This package exposes the reviewed Whereby video support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-video-whereby/manifest";
```
