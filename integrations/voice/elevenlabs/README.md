# @cognidesk/integration-voice-elevenlabs

ElevenLabs voice integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-voice-elevenlabs
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-voice-elevenlabs";
```

This package exposes the reviewed ElevenLabs voice support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-voice-elevenlabs/manifest";
```
