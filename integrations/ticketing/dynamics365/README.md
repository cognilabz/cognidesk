# @cognidesk/integration-ticketing-dynamics365

Microsoft Dynamics 365 ticketing integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-ticketing-dynamics365
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-ticketing-dynamics365";
```

This package exposes the reviewed Microsoft Dynamics 365 ticketing support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-ticketing-dynamics365/manifest";
```
