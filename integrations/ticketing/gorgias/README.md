# @cognidesk/integration-ticketing-gorgias

Gorgias ticketing integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-ticketing-gorgias
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-ticketing-gorgias";
```

This package exposes the reviewed Gorgias ticketing support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-ticketing-gorgias/manifest";
```
