# @cognidesk/integration-ticketing-pega-customer-service

Pega Customer Service ticketing integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-ticketing-pega-customer-service
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-ticketing-pega-customer-service";
```

This package exposes the reviewed Pega Customer Service ticketing support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-ticketing-pega-customer-service/manifest";
```
