# @cognidesk/integration-ticketing-oracle-service

Oracle Service ticketing integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-ticketing-oracle-service
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-ticketing-oracle-service";
```

This package exposes the reviewed Oracle Service ticketing support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-ticketing-oracle-service/manifest";
```
