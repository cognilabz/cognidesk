# @cognidesk/integration-email-gmail

Gmail email integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-email-gmail
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-email-gmail";
```

This package exposes the reviewed Gmail email support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-email-gmail/manifest";
```
