# @cognidesk/integration-email-mailgun

Mailgun email integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-email-mailgun
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-email-mailgun";
```

This package exposes the reviewed Mailgun email support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-email-mailgun/manifest";
```
