# @cognidesk/integration-ecommerce-shopify

Shopify ecommerce integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-ecommerce-shopify
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-ecommerce-shopify";
```

This package exposes the reviewed Shopify ecommerce support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-ecommerce-shopify/manifest";
```
