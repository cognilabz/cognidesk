# @cognidesk/integration-sms-twilio

Twilio SMS integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-sms-twilio
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-sms-twilio";
```

This package exposes the reviewed Twilio SMS support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-sms-twilio/manifest";
```
