# @cognidesk/integration-review-googleplay

Google Play review integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-review-googleplay
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-review-googleplay";
```

This package exposes the reviewed Google Play review support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-review-googleplay/manifest";
```
