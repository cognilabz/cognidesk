# @cognidesk/integration-marketplace-ebay

eBay marketplace integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-marketplace-ebay
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-marketplace-ebay";
```

This package exposes the reviewed eBay marketplace support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-marketplace-ebay/manifest";
```
