# @cognidesk/integration-marketplace-amazon

Amazon marketplace integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-marketplace-amazon
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-marketplace-amazon";
```

This package exposes the reviewed Amazon marketplace support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-marketplace-amazon/manifest";
```
