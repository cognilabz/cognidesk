# @cognidesk/integration-contact-center-amazon-connect

Amazon Connect contact center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-contact-center-amazon-connect
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-contact-center-amazon-connect";
```

This package exposes the reviewed Amazon Connect contact center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-contact-center-amazon-connect/manifest";
```
