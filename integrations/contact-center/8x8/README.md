# @cognidesk/integration-contact-center-8x8

8x8 contact center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-contact-center-8x8
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-contact-center-8x8";
```

This package exposes the reviewed 8x8 contact center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-contact-center-8x8/manifest";
```
