# @cognidesk/integration-contact-center-genesys-cloud

Genesys Cloud CX contact center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-contact-center-genesys-cloud
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-contact-center-genesys-cloud";
```

This package exposes the reviewed Genesys Cloud CX contact center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-contact-center-genesys-cloud/manifest";
```
