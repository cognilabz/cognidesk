# @cognidesk/integration-contact-center-ringcentral

RingCentral RingCX contact center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-contact-center-ringcentral
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-contact-center-ringcentral";
```

This package exposes the reviewed RingCentral RingCX contact center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-contact-center-ringcentral/manifest";
```
