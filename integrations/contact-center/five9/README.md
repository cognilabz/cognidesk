# @cognidesk/integration-contact-center-five9

Five9 contact center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-contact-center-five9
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-contact-center-five9";
```

This package exposes the reviewed Five9 contact center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-contact-center-five9/manifest";
```
