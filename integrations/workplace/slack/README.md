# @cognidesk/integration-workplace-slack

Slack workplace integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-workplace-slack
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-workplace-slack";
```

This package exposes the reviewed Slack workplace support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-workplace-slack/manifest";
```
