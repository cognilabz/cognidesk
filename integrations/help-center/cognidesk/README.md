# @cognidesk/integration-help-center-cognidesk

Cognidesk help center integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-help-center-cognidesk
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-help-center-cognidesk";
```

This package exposes the reviewed Cognidesk help center support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-help-center-cognidesk/manifest";
```
