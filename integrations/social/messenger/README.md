# @cognidesk/integration-social-messenger

Facebook Messenger social integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-social-messenger
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-social-messenger";
```

This package exposes the reviewed Facebook Messenger social support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-social-messenger/manifest";
```
