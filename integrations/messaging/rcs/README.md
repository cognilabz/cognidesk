# @cognidesk/integration-messaging-rcs

Google RCS for Business messaging integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-messaging-rcs
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-messaging-rcs";
```

This package exposes the reviewed Google RCS for Business messaging support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-messaging-rcs/manifest";
```
