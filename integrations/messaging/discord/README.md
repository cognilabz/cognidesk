# @cognidesk/integration-messaging-discord

Discord messaging integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-messaging-discord
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-messaging-discord";
```

This package exposes the reviewed Discord messaging support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-messaging-discord/manifest";
```
