# @cognidesk/integration-messaging-whatsapp

WhatsApp Business Platform messaging integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-messaging-whatsapp
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-messaging-whatsapp";
```

This package exposes the reviewed WhatsApp Business Platform messaging support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-messaging-whatsapp/manifest";
```
