# @cognidesk/integration-form-cognidesk

Cognidesk form integration package for Cognidesk support agents.

## Install

```bash
npm install @cognidesk/integration-form-cognidesk
```

## Usage

```ts
import * as cognideskPackage from "@cognidesk/integration-form-cognidesk";
```

This package exposes the reviewed Cognidesk form support workflow surface for Cognidesk. It is intentionally scoped to the provider operations declared by the package manifest and does not claim full provider API coverage.

For manifest-only usage:

```ts
import * as providerManifest from "@cognidesk/integration-form-cognidesk/manifest";
```
