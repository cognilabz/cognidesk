## Prerequisites

- **Node.js** 18+ or compatible runtime (Bun, Deno)
- **TypeScript** 5.0+ (recommended)
- **pnpm**, npm, or yarn
- An LLM provider account (OpenAI, OpenRouter, or compatible)
